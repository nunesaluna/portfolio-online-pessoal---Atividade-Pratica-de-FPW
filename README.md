# 🎓 Atividade Prática: Fundamentos da Programação Web 
	# 🌟 Meu Portfólio Pessoal

Este projeto foi desenvolvido como parte da disciplina de **Fundamentos da Programação Web**. 
O objetivo principal foi aplicar conceitos de estruturação semântica em HTML5, estilização moderna com CSS3 e interatividade básica com JavaScript.

> **🔗 Acesse o projeto ao vivo:** [[clique aqui](https://nunesaluna.github.io/portfolio-online-pessoal---Atividade-Pratica-de-FPW/)]

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout responsivo (Flexbox e Grid) e variáveis para o tema.
*   **JavaScript (Vanilla):** Lógica para troca de tema e interação com o menu mobile.

---

## ✨ Funcionalidades

- [x] **Design Responsivo:** Adaptado para celulares, tablets e computadores.
- [x] **Dark Mode:** Alternância entre tema claro e escuro para melhor conforto visual.
- [x] **Navegação Dinâmica:** Menu hambúrguer para dispositivos móveis.
- [x] **Formulário de Contato:** Funcional com simulação de envio e interface intuitiva.
